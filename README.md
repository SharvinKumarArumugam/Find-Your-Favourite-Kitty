# Find Your Favourite Kitty

First of all, create a new folder of the project then install all the dependency 

## Available Scripts

•	npx create-react-app paws-preferences
•	cd paws-preferences
•	npm install react-tinder-card


### For smooth animation I use 

•	npm install @react-spring/web --legacy-peer-deps
•	npm install react-tinder-card --legacy-peer-deps


### serve the page

npm start
